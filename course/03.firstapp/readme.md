## Requirements:
#### Core-js: (3.6.4)

```sh
$ npm install --save core-js@^3
```

#### Karma-jasmine-html-reporter: (1.5.1)

```sh
$ karma-jasmine-html-reporter@1.5.1
```

#### Jasmine-core: (3.5)

```sh
$ npm install jasmine-core
```

## Run Angular (default, or specific port)

```sh
$ ng serve
$ ng serve -p <port>
```
