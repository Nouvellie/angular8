# INSTALL
## Node
#### Download:

[Node.](https://nodejs.org)

#### Install:

- Node.js. (13.7.0)
- Npm. (6.13.6)

##TypeScript
#### Install: (3.7.5)

```sh
$ npm install -g typescript
```

## Angular CLI
#### Install: (8.3.23)

```sh
$ npm install -g @angular/cli
```

#### If errors:

```sh
$ sudo npm install --unsafe -perm -g @angular/cli
```

#### Check:

```sh
$ ng --version
```
